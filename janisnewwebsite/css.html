CSS

1. Vad är en regel i CSS och vad består den av?

En regel i css är en grundläggande komponent som används för att definiera hur HTML-element på en webbsida ska visas. Den består av två huvudkomponenter (selektor) och en (deklarationsblock). Selektorn är den del av CSS-regeln som identifierar vilka HTML-element eller elementgrupper som regeln ska tillämpas på. Den kan vara ett elementnamn, en klass, en ID. Ex. #my-id. Deklarationsblocket innehåller flera deklarationer som specificerar de stilattribut som ska tillämpas på de valda elementen. Varje deklaration består av ett egenskapsnamn och ett värde. Ex.






2. Vad är skillnaden mellan id och class? När bör man använda id respektive class?

Id är en unik identifierare för ett HTML-element. Det innebär att det endast kan finnas ett element med samma id i en HTML-dokument. Den användas när man behöver rikta in ett enskilt unikt element på sidan. Till exempel, om man vill ändra stilen för sidans huvudrubrik.



Class är en gemensam klassificering som flera element kan dela. Man kan använda samma klass på flera olika element på samma sida. Används när man vill tillämpa samma stil eller funktionalitet på flera element.
